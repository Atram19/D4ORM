from flax import struct
una libreria di machine learning basata su JAX (sviluppata da Google) per costruire modelli neuronali funzionali. In particolare, flax.struct serve per definire data class immutabili e compatibili con JAX.


@struct.dataclass
class MyState:
    step: int
    value: float

    MyState è ora una struttura immutabile: se vuoi cambiare un campo, devi creare una nuova istanza.

    Può essere usata come stato in un processo iterativo, come nel training di un modello.


    
partial ti permette di preimpostare alcuni argomenti di una funzione, creando una nuova funzione con meno parametri da specificare. È utile quando hai una funzione generale ma vuoi fissare alcuni valori per riutilizzarla facilmente.

from functools import partial

def moltiplica(a, b):
    return a * b

# Preimposto a = 2
doppio = partial(moltiplica, 2)

print(doppio(5))  # Output: 10
